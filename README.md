# Tweeter Project

Tweeter is a simple, single-page Twitter clone.


## Dependencies

- Express
- Node
- Nodemon
- Body-parser
- Chance
- Md5
- Mongodb

## Getting Started

-Install all dependencies (using 'npm install' command). \
-Run the development web server using the 'npm start' command.
